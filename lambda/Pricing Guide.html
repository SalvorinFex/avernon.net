Service Descriptions

Anti-Spyware/Anti-Virus

Unwanted Pop-ups and wallpaper changes are signs that your system might have spyware.  Includes installing several different Anti-Spyware programs and running scans with each of them.  Slow internet speed and programs crashing might indicate your computer is infected with a Virus.  Either updating your existing AV software or installing a new one is included with this service if it looks like there is an infection.

 

System Tune-up

Slow boot and shut down times as well as programs taking a long time to start are signs that the registry as well as other system level files need to be optimized.

This service includes installing and running a software suite that will speed up system load times and programs will start faster.

 

Software Install

Includes installing a program like Photo Editing, an Office Suite or Disc Burning utilities.

 

Email Account setup

If you have changed Internet Service Providers recently and are still using their Web Pages for your email, most likely they have what's called a POP service that can download your email into a client, like Outlook or Thunderbird.  From there you can back up and save important emails should you once again change ISPs.

 

Printer Setup

Waiting to get that new printer out of the box? Windows not seeing the printer anymore?  We can set it up for the first time or get it working again so you print out your quarterly statements. 

 

Wireless Network Setup

Many of today's computer related devices include wireless transceivers and can be setup to connect to your home or laptop computers.  These included printers, music players, television add-ons and game consoles.

Each device needs to be setup separately and connected through your existing wireless router.

 

Peripheral Upgrade

Needing a new sound card, video card or video capture device installed?

A shiny new Blu-Ray player would be able to kick out some serious HD.

We will crack open your computer's case and install it as well as the latest drivers from the manufacturer.

 

Data Back-up
(Device would be a external Hard Drive that you own or would need to buy)

This is a very important aspect of today's computer environment, your treasured pictures, music and documents need a safe place to be stored and this is the way to do it.  Once you have a portable hard drive we will create a back-up of your data and show you some simple steps to keep it updated and stored in, for example, a safety deposit box.

 

Data Recovery
(Device would be a external Hard Drive that you own or would need to buy)

Deleted files and folders can be recoverd if the drive still works.  When your system's hard drive goes bad there is a chance that the data can still be recovered.  If it still spins up and the computer sees it your data could still be there even after accidental formatting.

 

Computer Upgrade

Just need a little more RAM, perhaps a bigger hard drive or a complete overhaul.

You have finally decided that it's time to get your system up to date with some of the new technology you have been hearing about. 

We will discuss the options with you and see what of your old system can be transferred to the new and improved computer.
